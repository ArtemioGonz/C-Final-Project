# Personal Finance Manager - Presentation

This repository contains:

---

## Program Description

The **Personal Finance Manager** is a console application in C++ that allows you to manage personal finances by adding, deleting, and listing transactions. It also supports saving and loading data from CSV files, monthly summaries, budget management with alerts, and transaction search and sorting.
